## 📌 Visão Geral

Este repositório contém meu portfólio pessoal de desenvolvedor — uma vitrine dos meus projetos, habilidades técnicas e perfil profissional.

O objetivo é oferecer uma experiência limpa, rápida e moderna para recrutadores, clientes e colaboradores que desejam entender rapidamente meu trabalho.

## ✨ Destaques

- Interface moderna e responsiva
- Vitrine de projetos com descrição e links
- Seção “Sobre mim” com contexto técnico
- Seção de contato / links sociais
- Estrutura amigável para SEO
- Arquitetura frontend orientada a performance

## ⚙️ Como Executar

### Pré-requisitos

- Node.js (recomendado LTS)
- npm, yarn ou pnpm

### Instalação

```bash
# Clone o repositório
git clone https://github.com/joaobroggine/portfolio.git

# Entre na pasta do projeto
cd portfolio

# Instale as dependências
npm install
# ou
yarn
# ou
pnpm install
```

### Rodar localmente

```bash
npm run dev
# ou
yarn dev
# ou
pnpm dev
```

### Build para produção

```bash
npm run build
# ou
yarn build
# ou
pnpm build
```

### Preview da build de produção

```bash
npm run start
# ou
yarn start
# ou
pnpm start
```
